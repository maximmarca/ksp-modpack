# KSP Modpack

Pack de mods para jugar **Kerbal Space Program** en multijugador con DarkMultiPlayer.

## Requisitos

- **Kerbal Space Program 1.12.5** (cualquier tienda: Steam, Epic, GOG)
- Las versiones de los mods deben coincidir entre todos los jugadores.

## Instalación

1. Descargá este pack: botón verde **`Code`** → **`Download ZIP`**.
2. Extraé el ZIP. Adentro vas a encontrar una carpeta **`GameData`**.
3. Copiá **el contenido** de esa `GameData` dentro de la carpeta `GameData` de tu instalación de KSP.
   - Resultado final, por ejemplo:
     `.../Kerbal Space Program/GameData/DarkMultiPlayer/`
4. No reemplaces tu `GameData` entera — solo agregás las carpetas de los mods adentro.

## Mods incluidos

| Mod | Versión | Licencia |
|-----|---------|----------|
| DarkMultiPlayer Client | v0.3.8.5 | MIT |

## Conectarse al server

Una vez instalado, abrí KSP → botón **DMP** → **Add Server**.
Los datos de conexión (dirección y puerto) te los pasa Maxi por privado.

> El **player name** tiene que ser exactamente el que esté en la whitelist del server.

## Cómo agregar mods nuevos (para el que mantiene el pack)

1. Poné la carpeta del mod nuevo dentro de `GameData/`.
2. Actualizá la tabla de "Mods incluidos" de este README.
3. Commit + push. Los demás vuelven a hacer *Download ZIP*.

⚠️ Subí solo mods cuya licencia permita la redistribución.
